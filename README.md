## 概要
マルボロのパックコードを写真からスプレッドシートにインポートするためのGoogle Apps Script

## 準備
1.GoogleDriveに任意のスプレッドシートと、画像ファイル配置用フォルダを作成する。

2.スプレッドシート内から拡張機能＞Apps Scriptを選択する

3.任意のプロジェクト名とし、packcode_import.gsを貼り付ける

4.ソースコード内のTARGET_ID、FOLDER_ID、SHEET_NAMEを変更する

5.Apps ScriptのサービスからDrive APIを追加する

## 使い方
1.画像フォルダにパックコードを撮影した写真をアップロード（1枚の写真に複数パックコードが含まれていてOK）

2.スプレッドシートを開き、SHEET_NAMEで設定したシート名のシートを作成する

3.上のツール一覧に当インポートツールが出てきているので押下する

## 挙動
アップロードされた写真をGoogleDocumentに変換しOCR処理をし、処理後データをスプレッドシートにインポートする。

XXXX-XXXX-XXXXの形式になってない文言については削除する

処理が終わったらアップロード画像および変換したdocは削除する。（削除したくなければ該当ソースをコメントアウト）

## 注意事項
1時間クオリティなので多少のバグあり。たまに処理がループする時があるが多分90秒でタイムアウトする
